# Archive Index

이 폴더는 **활성 개발 문서에서 분리된 이력성 기록**만 보관한다.

## 포함 문서

- `incidents.md`  
  주요 이슈 원인/조치/재발방지 기록

- `sessions/`  
  날짜별 개발 세션 요약

- `reviews/`  
  시점별 프로젝트 리뷰 기록

## 정리 원칙

- 현재 운영/개발의 기준 문서는 `docs/development/`, `docs/reference/`, `docs/gpl-language/`를 우선한다.
- 중복 인덱스, 임포트 흔적 파일, 오래된 안내 README는 보관하지 않는다.
- 아카이브 문서는 “삭제 대신 이관”이 기본이며, 중복만 제거한다.
