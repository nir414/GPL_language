# Test_robot 프로젝트 구조 및 리뷰

**최종 업데이트**: 2025-12-09

## 프로젝트 개요

Brooks Automation Guidance 컨트롤러/PreciseFlex 로봇 제어 시스템. GPL(VB.NET 유사) 기반, 멀티 TCP/IP 통신, XML 데이터 영속성, Flash 수명 관리, 비동기 파일 저장 등 실전 배포 수준의 구조.

## 전체 모듈 구조

| 주요 모듈                | 책임/설명                          | 상태   |
|-------------------------|-----------------------------------|--------|
| Core_Main.gpl           | 시스템 진입점, 초기화, 메인 루프   | ✅ 권장 |
| Core_ErrorHandler.gpl   | 통합 로깅/예외 처리                | ✅ 권장 |
| Core_Utils.gpl          | 공통 상수/유틸리티                 | ✅ 권장 |
| Core_StringUtils.gpl    | 문자열 파싱/처리                   | ✅ 권장 |
| Data_XmlStore.gpl       | 인스턴스 기반 XML 저장소           | ✅ 권장 |
| Data_DatStore.gpl       | 인스턴스 기반 DAT 저장소           | ✅ 권장 |
| Data_AsyncSave.gpl      | 비동기 저장 큐 (범용)              | ✅ 권장 |
| IO_FileManager.gpl      | Flash 수명 관리, 안전 저장         | ✅ 권장 |
| IO_TcpCommunication.gpl | 멀티 TCP/IP 서버                   | ✅ 권장 |
| Robot_CeramicJig.gpl    | 세라믹 치구 제어                   | 🗑️ 제거됨(템플릿) |
| Robot_AirSolCylinder.gpl | 공용 액추에이터(포팅) - AirSolCylinderClass | ✅ 권장 |
| Robot_SimulatedAction.gpl | 테스트/시뮬레이션                | 📝 예제 |
